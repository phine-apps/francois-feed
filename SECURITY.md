# Security Policy

## Supported Versions

The following versions of francois-feed are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| v0.2.x  | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within francois-feed, **please do NOT open a public GitHub Issue**, as this would publicly disclose the vulnerability before a fix is available.

Instead, please use **GitHub's Private Vulnerability Reporting**:

1. Navigate to the [Security Advisories](https://github.com/phine-apps/francois-feed/security/advisories/new) page.
2. Click **"Report a vulnerability"** and fill in the details privately.

This ensures the report is kept confidential between you and the maintainers until a fix is released.

We will acknowledge your report within **72 hours** and provide an estimated timeframe for a fix. Please do not disclose the vulnerability publicly until a patch has been released and you have been notified.

Thank you for helping keep francois-feed secure!
